# Installed Wayne Skill registry

This is a capability inventory, not a route map. The isolated runtime has these
cross-agent Skill slugs available to a handoff packet:

- `wayne-mind-explode`
- `wayne-test-design`
- `wayne-plan`
- `wayne-work`
- `wayne-code-review`
- `wayne-verify`
- `wayne-ship`
- `wayne-compound`

A verdict, arrow-separated chain, external team, or unlisted slug is not a Skill.
