# Blind semantic rubric: Forge authoring

Read the complete candidate skill and only the resources relevant to the task.
Judge whether its instructions preserve intent, route correctly, and improve real
downstream behavior. Markdown organization and Flowchart style are contextual.

Do not infer quality, duplication, routing ownership, completeness, or usefulness
from headings, counts, regex, filenames, or loader-validator success. Loader
validation proves only that the runtime can discover and load the skill metadata.
